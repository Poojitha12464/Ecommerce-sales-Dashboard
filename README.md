# Power BI Ecommerce Sales Dashboard

## Overview
This project is an interactive Ecommerce Sales Dashboard created using Power BI.  
The dashboard helps analyze sales performance, revenue trends, product performance, and regional sales distribution using visual analytics.

---

## Objectives
- Analyze total sales revenue
- Track monthly sales trends
- Identify top-performing products
- Compare category-wise revenue
- Analyze regional sales contribution
- Build an interactive business dashboard

---

## Tools & Technologies Used
- Power BI Desktop
- CSV Dataset
- Data Visualization
- DAX Measures

---

## Dataset Information
The dataset contains:
- Product Details
- Categories
- Revenue
- Orders
- Units Sold
- Regions
- Monthly Sales Data

---

## Dashboard Features

### KPI Cards
- Total Revenue
- Total Orders
- Total Units Sold

### Visualizations
- Monthly Revenue Trend (Line Chart)
- Revenue by Category (Bar Chart)
- Regional Sales Distribution (Donut Chart)
- Top Products Analysis (Bar Chart)
- Revenue vs Units Sold (Scatter Plot)

### Interactive Features
- Region Filter
- Month Filter
- Category Filter

---

## Key Insights
- Identified highest revenue-generating categories
- Analyzed monthly business growth
- Compared regional sales performance
- Detected top-selling products
- Evaluated sales distribution patterns
